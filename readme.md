# Code 'n Craft Web App
Features the tools & timers used for Code 'n Craft events. Made with D3.


## Live Site
https://amitthekumar.gitlab.io/code-n-craft/


## Local Development

### Project Setup

``` shell
$ npm install
```

### Run

``` shell
$ npm start
```
This starts a browserSync server. View the site at http://localhost:3000.
